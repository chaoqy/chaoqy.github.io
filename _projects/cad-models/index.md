# Linear Induction Motor (Smith Engineering Hyperloop)
I worked on modeling a double-sided 3-phase, 9-coil linear induction motor, along with designing a mounting system and a custom winding rig to support it. While developing the mount, I ran static analyses to make sure each component could handle expected loads safely. My initial design ended up being heavily overbuilt (with a factor of safety above 20), so I iterated on it to reduce material usage while still maintaining structural integrity.

# RoboMaster Robot (Queen's Knights Robotics Team)
For the RoboMaster North America competition, I designed the robot’s frame and hopper assembly. I focused on making the hopper as reliable as possible, so I built a gravity-fed pellet storage system with angled walls to guide pellets smoothly into the turret. This helped prevent jamming and ensured consistent feeding during operation.
